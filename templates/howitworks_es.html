Cómo Funciona

"Encuéntrame Un Whiskey" usa una base de datos de más de 2.000 whiskeys. El código usa Natural Language Processing (NLP) para extraer los sabores.

Usamos los sabores más comunes y etiquetamos al whiskey con ellos basado en la descripción

Como los whiskeys pueden ser descritos de tantas maneras, usamos una técnica de asociación de palabras para encontrar sinónimos. Esta técnica es Word2Vec

Word2Vec

Word2Vec es una técnica de NLP. Usa un modelo de neural network para aprender la asociación de palabras de un cuerpo te texto. Una vez entrenado, este modelo puede detectar sinónimos.

ADespués de entrenar nuestro modelo con las descripciones y otras opiniones de whiskeys que fueron extraídas de páginas web, tenemos nuestros sinónimos.

Por ejemplo, una de nuestras etiquetas de sabor es la palabra turba, para la que el código encuentra:
